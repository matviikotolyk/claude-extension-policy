Privacy Policy for Claude Prompt Improver

Last updated: 10/25/2024

1. Data Collection

This extension collects the following data:

  • User prompts entered on claude.ai for the purpose of improvement

2. Data Usage

The collected data is:

  • Sent to Groq API for prompt improvement
  • Not stored permanently
  • Not used for any other purpose

3. Data Sharing

We share data only with:

  • Groq API for prompt processing
  • No other third parties

4. User Rights

Users have the right to:

  • Disable the extension at any time
  • Control when prompts are sent for improvement
